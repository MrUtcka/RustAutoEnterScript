This is simple AutoEnter Script for Rust


Dependence:
```
pip install schedule
pip install time
pip install keyboard
```

To use you need:
1. Set a key to open the console: f1
2. Set server ip(without connect): mrutcka.gg
3. Set server wipe time(in your local time): 00:00
4. Set the server wipe day of the week: monday
5. Synchronize your computer time with online time(https://time100.ru or others)

After that you can press RUN!!!
